OnMax

OnMax was developed to solve the issue enterpise was having in regards to the slow onboarding practices that dragged the process out of multipule days. I was taked with creating a solution.
Utilizing VBA and Excel I created a set of macros that streamlined and automized the process for onboarding personal to create the nessisary documents that they needed to complet there daily tasks.
OnMax has the ability to import data from the windows clipboard, and then the end user can choose how they would like the data formated. The key benifit of this software was that the orginal data was around 100 rows of 50 coloms each, and the export of any one specific document would only need 4 - 8 of thoughs coloms, but each export needed diffrent information. OnMax digested this information and exported the nessisary files in less than a 10th of the time that it could be done by hand, saving time money and fatigue. This software is still currently inuse by Enterprise's Onboarding department, a second version is also currently in development.
